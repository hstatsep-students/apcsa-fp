One Dimensional Arrays

What are arrays?

Arrays are used to store large amounts of information without needing to set up separate variables for each piece of information. Arrays are objects and can hold any kind of data, including primitive types like ints, other class data types like Strings, or even more arrays. To declare and initialize array, use datatype[] name = new datatype [//num of space you want memory to allocate]. To change the elements inside arrays, use nameofarr[//index of the element you want to chnage] Like Strings, arrays have indexes which start from 0 to end at number of elements -1
Elements 0 0 0 0 0 0
Indexes 0 1 2 3 4 5
Using an index value outside of this range will result in an ArrayIndexOutOfBoundsException. After creating the array, we CANNOT CHANGE THE AMOUNT OF INDEXES. Arrays can hold class data type like how they store primitive types. To call methods on object in arrays we can just access element from the array using name[//index].method().

What are other ways arrays use Algorithms ?

Ways Algorithms are used in arrays:
  • insertion: when we insert we need to move or shift other elements to make space for new so we shift using an loop For example,if we want to insert something in index 3:
    for (int i = list.length - 1; i > 3; i--){
    list[i] = list[i - 1]; }
     NOTE: the last element in the initial array will be removed
  • deletion: when we delete we want to remove a value so we need to shift elements. For example, remove an item on index 2:
    for (int i = 2; i < list.length - 1; i++){
    list[i] = list[i + 1];}
  • shifting elements
    1. To shift every element one place left:
      int temp = list[0];
      for (int i = 0; i < list.length - 1; i++){
      list[i] = list[i + 1];}
      list[list.length - 1] = temp;
    2. To shift every element to right :
      int temp = list[list.length - 1];
      for(int i = list.length -1; i > 0; i--){
      list[i] = list[i - 1];}
      list[0] = temp;
  • reversing elements: when reversing an element, we are swapping the element from the start of the list with the end using a dummy variable to enable the swap. We stop halfway or else it would swap everything back to original position
    for (int i = 0; i < list.length / 2; i++){
    int j = list.length - i - 1; //index of element in end half
    int temp = list[i];
    list[i] = list[j];
    list[j] = temp;}

For-each Loops

What is the for-each loop?

This is an enhanced for loop where we do not have to declare a new int and comparing it to the length of an array in ofer to traverse The format of this string: for (type e : array){//code} This code basically translates tha for each item e of datatype type in array, perform some action. The e is the enhanced for loop variable

What the difference between for loops and for-each loops?

A for loop says “do something a certain num of time” while the for-each loop says “do this with every item in the list” which makes it easy to avoid off-by-one errors with the indexes.

What are the limitation of for-each loops?

  • cannot change the contents of the array when it is run
  • it does not allow us to access the index of the element it is currently accessing, perform operations on more than one element of the array, or traverse more than one array simultaneously. This means that any algorithms that compares or shift values cannot be used

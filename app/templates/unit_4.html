Loops: Repeat commands.

How do while loops work?

While loops have two components: the condition, and the code to execute if the condition is true, just like if statements. The difference is that a while loop will keep executing the code over and over as long as the condition is true:
while (//condition){
 //commands that will gets repeat
}
 While loops will only start if the condition is true to begin with and will only be stopped when the coniditon is FALSE. Be aware that the variables made inside a loop or if statements is only limited inside.

How do for loops work?

For loops works with counting the numbers of repeats The for loops contains a loop control variable, which functions as the counting variable and controls the loop, is central to the function of a for loop.
for (int i = 0; i < 10; i++){
 //commands that will gets repeat
}
 The header of the for loop:
  • i = 0: initializes the loop. It runs once on the first iteration of the loop to set the starting value for the loop control variable.
  • i < 10: tells the loop when to stop. This boolean statement is checked before every iteration of the loop after the increment/initialization statement is run. The loop exits when it is false.
  • i++: tells the loop what to count by. More specifically it increments or changes the loop control variable in a set way. After the loop has finished running, the control variable is incremented in preparation for the next iteration of the loop.

So when should use the while loop or the for loop?

We used for loops when we have a definite starting and stopping point, or when we know exactly how many times we want to repeat something. We used while loops when we do not know how many times the commands will get repeated.

What about nested loops?

Nested loops are loops inside a loop. The scope of the loop control variable for the outer loop is everything within that loop, including inside the inner loop. This means we can reference both of our control variables from inside the inner loop, and use the outer loop control variable in the header for our inner loop. Use a trace table to keep track. Algorithms are patterns or instructions used in programming. Algorithms must have operations that can be done by a computer, produce a result, and stop in a finite amount of time.

Unit 8: 2D Arrays

About 2d Arrays

What are 2d arrays?

2d arrays are arrays that have rows and columns. Rules that apply to normal array applies to 2D arrays:
  • Store one type of data
  • Indexed from 0 to length - 1
  • Used to store large amounts of data
  • 2d array rows can have different length

How to set up 2d arrays?

To declare 2d arrays use: datatype[][] grid = new datatype[//num of rows][//num of column];. To access the elements in 2d array, use: grid[//row][//colums].

How to Loop through 2d arrays?

There are two ways to loops through 2d arrays:
  • Row-major: access rows first
    for (int r = 0; r < grid.length; r++){ //accesses rows
     for (int c = 0; c < grid[0].length; c++){ //accesses columns
      System.out.print (grid [r][c] + " \t"); //prints elements on same line
     }
     System.out.println( ); //skips to next line
    }
  • Column-major: access columns first
    for (int c = 0; c < grid[0].length; c++){ //accesses columns
     for (int r = 0; r < grid.length; r++){ //accesses rows
      System.out.print (grid [r][c] + " \t"); //prints elements on same line
     }
     System.out.println( ); //skips to next line
    }
